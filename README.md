## What is this?

Simple Pong Game where Naruto yells DATTEMAYO! and SASUKE! on random when he hits the ball

## Goal

Hit the Ball with Paddles

## Controls

Use W and S if you are Naruto and UP and DOWN key if you are Sasuke
